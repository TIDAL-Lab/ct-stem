A climate scientist wrote this computer program to measure air temperatures at a weather station: 
repeat 20 times:
   pause-minutes(30)
   record-temperature
Item 1: Which of these plots shows her measurements?
The data collected by the air temperature program is stored in a file called temperature-readings.dta. To analyze the data, the climate scientist wrote another program, shown below. 
for-each temperature-data-point in temperature-readings.dta:
   if temperature-data-point < 12° AND temperature-data-point > 11°:
      print temperature-data-point
Item 2: What does this program do?
  1. Prints the first temperature reading less than 12°
  2. Prints all temperature readings
  3. Prints all temperature readings bewteen 11° and 12°
  4. Prints all temperature readings greater than 12° or less than 11°
To display the data in temperature-readings.dta in a graph, the climate scientist wrote the following program: 
for-each temperature-data-point in temperature-readings.dta:
   if temperature-data-point > 12°:
      set plot-color red
   else if temperature-data-point < 11°
      set plot-color blue
   else 
      set plot-color black
   plot temperature-data-point

A team of engineering students is building a robot for a contest. To win, they must program their robot to move around a grid while avoiding obstacles completely on its own. The robot moves exactly one square at a time either up, down, left, or right.

The team started by writing this program to move their robot.

   move-right
   move-up
   move-right
   move-right
Item 4a: After running the program, which square will the robot stop on.
